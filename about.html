Experience Developing This Website

This website was developed as part of my Assignment in web development. The website features a home page, a shop page with categories for men's and women's clothing, a cart page, and an about page. The development process involved the following steps:

  • Designing the layout and structure of the website using HTML.
  • Styling the website with CSS to ensure a visually appealing and responsive design.
  • Implementing interactive features using JavaScript, such as adding items to the cart and dynamically updating the cart's content.

Technologies Used

  • HTML5: For structuring the content of the website.
  • CSS3: For styling and layout of the website, ensuring responsiveness and a modern look.
  • JavaScript: For adding interactive elements, such as cart functionality and dynamic content updates.
